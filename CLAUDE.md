## Common Commands
- Install dependencies: `npm install`
- Run dev server: `npm run dev`
- Build: `npm run build`
- Lint: `npm run lint`
- Prisma generate: `npx prisma generate`
- Prisma migrate: `npx prisma migrate dev`
- Lint: `npm run lint`
- Type check: `npm run typecheck`
- Run tests: `npm test`
